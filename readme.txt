続いて，G説明します．
メンバーは総勢20名で，園芸製品における先行技術開発を担当しています．
-Gはそれぞれの担当製品の開発を行っている-Gと異なり，製品開発の前段階として、新しい技術の検討や検証を行う役割を担っています。
技術の実用化の目途が立つと、各製品の担当部署での製品開発につながり、最終的にユーザへと届けられます。また、市場や顧客ニーズを踏まえながら開発を進めており、ユーザが求める製品を実現するための技術開発と、他社にはない新しい価値の創出に取り組んでいます。

続いて、業務内容の一例について説明します。
まず、刃物・アクセサリ開発では、せん定ハサミの刃やアタッチメントなどを対象に、用途に応じた設計・開発を行っています。
また、Uハンドル機やヘッジトリマ、ブロアといった製品に関連するアクセサリについても開発を行っています。
次に、新製品開発は、これまでにない新しい価値を持った製品の検討・開発を行っています。
さらに、振動・騒音対策として、集塵機やブロアの静音化など、作業時の快適性向上に向けた技術開発にも取り組んでいます。

先行技術開発では大きく4つの観点を重視しています。
1つ目は特許調査で、既存技術の把握や新規性の確認を行っています。
2つ目は規格調査で、安全性や法規制を踏まえた開発を進めています。
3つ目は他社調査で、競合製品の動向や技術レベルを分析しています。
4つ目は製品評価で、実際の使用を想定した検証を行い、技術の有効性を確認しています。
これらを通して、実用性と独自性のある技術開発に取り組んでいます。
また、部署の雰囲気としては、疑問や気づきを気軽に相談できる環境があり、メンバー同士で意見交換をしながら業務を進めています。
